The What?

Gravatar is a "globally recognized avatar API used to attach an avatar as representation of you online."

The Why?

  • Prevents inappropriate images on your website from users
  • Eliminates redundancy in uploading a user image on every social platform
  • Eliminates the need to code avatar image uploading in applications
  • Eliminates the need to add cloud databases such as Amazon Web Services (S3) to the application

Only works with websites that uses the Gravatar plugin

Create myContacts

Open the Terminal(Mac) or Command Line (PC)

rails new myContacts

cd myContacts

rails generate scaffold Users firstname:string lastname:string emai:string phonenumber:string

rake db:migrate

Within your project go to config/routes.rb

Delete everything that has ''#'' next to it

root 'user#index'

Within your project go to the Gemfile

gem 'bootstrap-sass', '~> 3.2.0.2'

Within the Terminal(Mac) or Command Prompt(PC)

bundle install

Within your project go to assets/stylesheet/application.css

Rename the file application.css.scss

At the end of the file add these two lines of code

@import “bootstrap-sprockets”;

@import “bootstrap";

Within your project go to app/helper/application_helper.rb


def gravatar_url(user)
	gravatar_id = Digest::MD5::hexdigest(user.email.downcase)
	default_url = "http://www.francaisasydney.com/images/avatars/gallery/woman1.jpg"
	url = "http://gravatar.com/avatar/#{gravatar_id}.png?s=128&d=#{CGI.escape(default_url)}"
end
